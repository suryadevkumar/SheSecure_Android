## Features

* User signup and secure OTP-based login
* One-tap SOS alert system
* Add and manage trusted contacts
* Share live location in real time
* Emergency helpline numbers
* Chat with counsellor (request-based and secure)
* Crime reporting to alert other users
* Location history tracking
* Safety tips for user awareness
-------------------------------------------------------------------------------------------------------------------------------------------------------

## Chat with Counsellor ‚Äì How It Works

* The user fills a small request form describing their concern or feeling
* The request is sent to a counsellor
* After the counsellor accepts the request, a private chat session starts
* This process ensures privacy and proper support
-------------------------------------------------------------------------------------------------------------------------------------------------------

## Report Crime ‚Äì Community Safety

* Users can report crimes or unsafe activities through the app
* Reported data helps other users stay alert in that area
* This feature improves safety awareness for the community
-------------------------------------------------------------------------------------------------------------------------------------------------------

## How to Use the App

1. Sign up and create a new account
2. Log in using email and OTP verification
3. Add trusted contacts
4. Use SOS in emergency situations
5. Share live location when needed
6. Report crimes to help other users stay safe
